
# Azure Cloud Storage AND Database Project

This project demonstrates core Azure services using the Microsoft Learn sandbox, including:

- Azure SQL Database
- Azure Blob Storage
- Azure File Shares (SMB)
- Azure Table Storage
- Azure Cosmos DB

All screenshots were taken during hands‑on deployment and configuration.

## Summary

This project demonstrates:

- Creating and configuring Azure SQL Database
- Managing Blob Storage containers and directories
- Connecting and working with Azure File Shares
- Using NoSQL data with Azure Table Storage
- Querying JSON documents in Azure Cosmos DB

All work was done using the free Microsoft Learn sandbox.
